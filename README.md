# nemotron-asr-local-streaming-demo

Local & sovereign demo scripts for **NVIDIA Nemotron ASR Streaming (0.6B)** - offline transcription + a live-microphone Gradio app. Run English speech-to-text fully on your own GPU, without a cloud API.

## Full step-by-step guide

The complete walkthrough (setup, model download, streaming, live demo) is on my blog:

**[Install NVIDIA Nemotron ASR Streaming Locally - Step-by-Step Guide](https://ai-box.eu/en/news/install-nvidia-nemotron-asr-streaming-locally-step-by-step-guide/2530/)**

## What's inside

- `transcribe.py` - offline transcription of a single audio file
- `asr_gradio_app.py` - live microphone transcription as a small Gradio web app
